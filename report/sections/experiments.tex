\section{Experiments \& Results}

\subsection{Experimental Setup}

The training on the Hopper-v4 environment lasted for 5000 episodes, this number was empirically found to be suffiecient for high-variance algorithms like REINFORCE in order to converge ensuring an exhaustive comparison.

All models in this environment were trained using the Adam optimizer wiht a fixed learning rate of $10^{-3}$ and a discount factor $\gamma = 0.99$. 

Furthermore, to account the stochasticity of policy gradients methos, for each configuration we used $3$ independent run using different random inizializator seeds.

All the policies were tested on $50$ test episodes.
